# AI Operating Mode

Este projeto utiliza agentes de IA como auxiliares de desenvolvimento.

Regras de operação:

- A IA NÃO é arquiteta do sistema
- A IA NÃO decide stack
- A IA NÃO cria padrões novos
- A IA executa tarefas dentro dos limites documentados

A função da IA é acelerar a execução,
não redefinir decisões.
