# Design System

## Filosofia

- Visual corporativo
- Legibilidade > estética
- Cores com semântica clara

## Paleta Base (Tailwind)

- Primary: indigo
- Secondary: sky
- Success: emerald
- Warning: amber
- Danger: rose
- Info: blue

## Status de Entrega

- PENDING → info
- IN_ROUTE → info
- DELIVERED → success
- DELAYED → warning
- FAILED → danger

## Componentes Padronizados

- Button: primary | secondary | ghost | destructive
- Badge: por status
- Card: rounded-xl, shadow-sm
